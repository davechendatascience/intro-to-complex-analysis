# Lecture 1: The Basics

## Theory
- **Complex Numbers Definition**: $\mathbb{C} = \{ x + iy \mid x, y \in \mathbb{R} \}$.
- **Real Part**: $\text{Re}(z) = x$.
- **Imaginary Part**: $\text{Im}(z) = y$.
- **Conjugate**: $\bar{z} = x - iy$.

## Theorems
- `Complex.add_conj`: $z + \bar{z} = 2\text{Re}(z)$.

## Exercises
- Prove that $z + \bar{z} = 2\text{Re}(z)$.
